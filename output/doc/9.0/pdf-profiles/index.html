PDF Profiles

User Guide / PDF Output

Prince supports optional PDF profiles, which can be selected using the --profile option on the command-line interface, or the setProfile method if using a server wrapper, or the PDF.profile property in JavaScript.

Currently the only available profile is PDF/A-1b. Future releases of Prince will add support for PDF/A-1a (tagged PDF) and PDF/X.
