prince-flow

Syntax

prince-flow: normal | static(name, [ start | current ]?)

Default

normal

Inherited

No

Comments

This property is used to remove elements from the normal flow of the document and place them in a page margin box to create running page headers and footers. See the documentation for Page Headers and Footers for more details.

Note that flow can be used as a shorthand for this property.
