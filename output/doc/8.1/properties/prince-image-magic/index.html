prince-image-magic

Syntax

prince-image-magic: none | jpeg-quality(percent) | snap-to-integer-coords

Default

none

Inherited

Yes

Comments

When a JPEG quality percentage value is specified for this property, Prince will recompress JPEG images at the specified quality level to save space when embedding them in the PDF output.

The snap to integer coords value can be used to avoid blurring of images in some PDF viewers.
